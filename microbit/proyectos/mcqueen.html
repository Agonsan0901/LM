McQueen Control Remoto

Descripción

Controla un robot McQueen usando el micro:bit como control remoto. Usa la inclinación del micro:bit para dirigir el robot en diferentes direcciones.

Materiales necesarios

  • 2 micro:bits
  • Robot McQueen
  • Baterías

Funcionamiento

Un micro:bit lee el acelerómetro y envía comandos por radio. El otro micro:bit (en el robot) recibe las instrucciones y controla los motores para mover el robot adelante, atrás, izquierda o derecha.

Conceptos aprendidos

  • Control de motores
  • Comunicación inalámbrica
  • Lectura del acelerómetro
  • Robótica básica
  • Mapeo de valores
  • Control remoto
